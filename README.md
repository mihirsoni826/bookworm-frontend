This React app serves as the frontend for the [Bookworm Backend API](https://github.com/mihirsoni826/bookworm-backend).


# Try it out

[Bookworm](https://bookworm-frontend-c2bc1357ec03.herokuapp.com/)

# Run locally

Create a `.env` file in the project root with the following variable:

`VITE_BOOKWORM_INTERNAL_API_BASE_URL` - The base URL with the context path of the Bookworm Backend API

## Install dependencies

```bash
npm install
```

## Run development server

```bash
 npm run dev
```

# Tech Stack
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)
- [Typescript](https://www.typescriptlang.org/)
